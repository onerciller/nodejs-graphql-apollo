## About

The API service is built with Nodejs, Graphql and latest stable versions of other npm packages. 

## Getting started

### Development

- Install npm modules:
  ```
  yarn install
  ```

- Create database migration
  ```
  yarn db:migrate
  ```
  
- Create database seed
  ```
  yarn db:seed
  ```

- Start server watch 
  ```
  yarn watch
  ```
